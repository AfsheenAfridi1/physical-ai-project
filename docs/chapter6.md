# Chapter 6 — Unity and Digital Twins

Unity is used to create high-fidelity digital twins for human-robot interaction.

## 6.1 What Is a Digital Twin?

A digital twin is a virtual replica of the real robot and environment.

## 6.2 Human–Robot Interaction

Unity allows simulation of:

Gestures

Handovers

Human movement

High-quality visualization
